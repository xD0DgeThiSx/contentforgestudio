SECTION 01
// SECTION — BUSINESS & STRATEGY
Prompts 1–5
01 Business Plan Summary
COPY-READY PROMPT
Act as a startup strategist. Summarize this business idea in 3 bullet points (problem, solution, unique edge). Keep it concise, like an investor pitch.
EXAMPLE IN ACTION
Act as a startup strategist. Summarize SwiftCart (same-day delivery app for indie retailers) in 3 bullets (problem, solution, unique edge) + 1 milestone (12-month goal) + 1 risk with mitigation. Keep investor-tight.
Real use: Used this to prep a cold pitch — kept me crisp and got immediate interest.
02 Competitor Analysis
COPY-READY PROMPT
Generate a competitor comparison table for [business name/type]. Include pricing, features, and one-sentence advantages/disadvantages.
EXAMPLE IN ACTION
Create a comparison table for Notion, Evernote, Obsidian, OneNote, Apple Notes with: price, core use case, offline support, AI features, switching friction. Add a 1-sentence pro & con per tool.
Real use: Built this for a client's launch deck — became the slide that sealed funding.
03 Offer Clarity Boost
COPY-READY PROMPT
Take this rough offer description and make it sharper and easier to understand for a non-expert audience.
EXAMPLE IN ACTION
Rewrite the Inbox Zero Sprint (30 days) coaching offer in 5 bullets as Feature → Outcome (+ metric). Include who it's for / not for and a risk-reversal line.
Real use: Turned a fuzzy coaching package into a sharp, premium offer.
04 SWOT Analysis
COPY-READY PROMPT
Act as a business consultant. Create a SWOT (Strengths, Weaknesses, Opportunities, Threats) for [my company name or industry]. Make it sound professional but brief.
EXAMPLE IN ACTION
Do a SWOT for BodegaCloud (SaaS for neighborhood grocers). 3 bullets each and a "So What" line: one decision this SWOT pushes us toward.
Real use: Used for a board update — kept discussion on the few variables that matter.
05 One-Liner Generator
COPY-READY PROMPT
Craft 3 bold, catchy one-liners to describe my business. Keep each under 10 words. Make them sound modern and memorable.
EXAMPLE IN ACTION
Generate a one-liner for PaddlePath Kayak Tours: [Audience] + [Problem] + [Solution] + [Proof] in under 20 words. Provide 3 options.
Real use: Became the headline we used across LinkedIn and the website.

// SECTION — MARKETING COPY & ADS
Prompts 6–10
06Landing Page Hero Copy
COPY-READY PROMPT
Write a bold, clear hero section headline + subhead for a landing page selling [product/service]. Focus on benefits. Make it emotionally engaging.
EXAMPLE IN ACTION
Write hero headline + subhead + CTA for BreezeCharge (EV home charger). Angle: renters with garages. Constraints: <10-word headline, mention 30-min install, CTA "See if your home qualifies."
Real use: A/B test winner — +22% hero click-through.
07Instagram Ad Hook
COPY-READY PROMPT
Write 3 scroll-stopping first lines for an Instagram ad promoting [offer]. Each should fit within 1 sentence and spark curiosity.
EXAMPLE IN ACTION
Generate 10 IG hook lines for a cold-plunge tub brand. Mix pattern interrupts, curiosity gaps, and benefit bombs; each 8 words or fewer.
Real use: Hook lines that didn't feel ad-y; CPM dropped 12%.
08AIDA Product Description
COPY-READY PROMPT
Use the AIDA copywriting formula (Attention, Interest, Desire, Action) to sell this product: [details]. Keep it 100–150 words.
EXAMPLE IN ACTION
Write an AIDA blurb for GlowDuo Vitamin C + Niacinamide Serum. Include derm-safe claims, evidence (study or %), and application steps. Cap at 160 words.
Real use: Lifted PDP add-to-cart 9% for a skincare SKU.
09Call-to-Action Builder
COPY-READY PROMPT
Give me 5 different call-to-action phrases for [goal], each using a slightly different emotional appeal (urgency, curiosity, community, etc.).
EXAMPLE IN ACTION
Give 12 CTA lines for a B2B demo booking. Group by Low-Friction / Medium / High-Commitment. No "Learn more."
Real use: Replaced weak CTAs; opt-ins rose 17%.
10Headline Variants
COPY-READY PROMPT
Give me 5 punchy headline options for a product targeting [audience], each using a different persuasion angle (fear, benefit, FOMO, proof, etc.).
EXAMPLE IN ACTION
Generate 10 headline options for PeakPack Daypack (lightweight hiking backpack). Mix curiosity, benefit, proof, and urgency; each 8 words or fewer.
Real use: Doubled trial activation rate after re-framing the first send.
